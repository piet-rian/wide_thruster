# Wide Thruster

RimworldのOdyssey(DLC)における、グラヴシップ用のスラスターを追加する

- サイズは縦2横3
- 最大接続数は2
- 若干ではあるが発射品質の向上に寄与する

## MID-SAVE

- 途中導入
  - 問題なし
- 途中で除去
  - 除去後の一回目だけエラーが出るがそれ以降は問題なし

## CONFLICT

多分競合はないはず
